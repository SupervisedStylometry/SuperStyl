 GHy Christen schaer // eerbaer neemt uwen roep klaer // wel waer Om verblyden hier naer // aldaer Geen scheyden en wert meer Als rechte Israelijten, Volgende Christus leer Laet ons belofften quijten Dat wy vaten ter eer, Oorbaer tot elcken keer., Wesen vanden Huys Heer Dat door goet gerucht // bewucht De blinde met vreucht // tot deucht En op dat ghy meucht // vol jeucht wassen in Christo seer. Wast nu ter tijdt // met vlijt Als rancken wijt // int krijt Op dat ghy sijt // bevrijt Van Gods tooren vertsaecht Die hem sal openbaren So Paulus gewaeght, Op alle boose scharen Die na Godt niet en vraecht Tsal noch worden beklaeght Wie in hem geen vrucht en draecht Sal hy na schrifs bedien // af snien Dus wilt wel in tien // toe sien En de boosheyt vlien // verbien Sal hy, die Godt behaeght, De boosheyt quaet // verstaet, Niet metten praet // maer daet Met goeden raet // na gaet Christum den Herder fijn Die ons heeft genesen. met Oly, en met wijn En veel liefde bewesen Als onsen Medecijn Hy droech al ons gepijn Deur zijn woorden devijn, Ghenas na schrifs vertel // seer snel Israel // oock wel Van der Slangen fel // tgequel Was doodelijck fenijn Tfenijn seer bloot // was groot Dat de Slangen hoot // uyt schoot Maer deur Christus doot // minjoot Is dat tot niet ghebracht, Sathan die heeft verloren Zijn gheweldige macht Daerom is hy vol toren Soeckende dach en nacht Laet ons houden goe wacht Het uytvercoren gheslacht Gaet hy met behent // ontrent, Als bekent // die schent Op dat hy verblent // afwent Van Godt hooghe gheacht Gaet ongestoort // doch voort Door de enge Poort // soot behoort En maeckt geen discoort // int woort Als eerlijcke leen, En wilt niet tegen stryden Gelijck de Phariseen Die den Heere benyden, En verstieren zijn reen Dat was meest dat zy deen Want zy waren van beneen, En hebben den Heylandt // vayljant Aen een kruys gespant // met schandt En dien hy sant // int lant Die hebben zy oock bestreen Sathan bevecht // Gods knecht Die nu tonrecht // afleght Soo ons Job seght // die slecht, God vreesde en zijn ghebodt Zijn wijf die hem wel kende Die heeft hem oock bespot En sprack in zijn ellende Blijft ghy noch vast by God Job die antwoorde tot Haer, ghy spreeckt als wyven sot Want hebben wy dat goet // seer vroet Met overvloet // men moet Oock tegenspoet // onsoet Ontfangen voor ons Lot.
