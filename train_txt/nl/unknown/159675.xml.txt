 LAet ons verblijden // met vreesen en beven Boosen schijn mijden // want ons is gegheven In onse lijden // int begin der tijden, Macht om te leven. In sondt gheboren // als de canenijten, En gantsch verlooren // ginck met ons versmijten Al soud wy smoren // niemant can ons hooren Priester Levijten. Stramen en wonden // waren al ons leden, Doot in der sonden // t'hert was onbesneden, Onreyne wonden // onghesouten gronden, Godloose zeden. Bloot ende naken // lagen wy met schanden, In alle saken // was niet voorhanden, Maer Godt liet maecken // rocken wie sal 't laecken, Van ons vyanden. En ginck ons proncken // met spansels oorringhen, Hy heeft ons schoncken // sijden costel dingen, Van gout wy bloncken // int riet gelijc voncken Cierlijck wy ginghen. Tot alle nacy // is dees faem gheresen, Van dese gracy // worden wy ghepresen, Maer wy och lacy // hoe van estimacy, Wort hier bewesen. In die Propheten // leeftment overvloedich, Hoe't wort verweten // Israhel stoutmoedich Dat sy vergheten // hoe sy had gheseten, Onreyn en bloedich. Alle die dieren // cranen ende swalven, Holt sijn manieren sijner comste halven, Maer te regieren // mijn volck recht te stieren, Helpter gheen salven. Kermen en clagen // hoe God heeft verheven Na sijn behaghen // Israhel van leven, Maer sy ontsagen // weldaet nochte plagen, Teghen te streven. Ossen en muylen // kennen hare Heeren, Maer ick moet huylen // mijn volck wil niet leeren, Het brist als guylen // t'graeft malcander cuylen Om te verheeren. Priesters en Heeren // die't volc souden straffen Zijn stom in't leeren // honden die niet blaffen T'goet sy verkeeren t'boose sy eeren, Terwe moet kaf zijn. Soud ick niet wreken // sulcke boose daden, Ick ginck uytreecken // alle mijn ghenaden, Met loose treecken // met hoveerdich spreken Sy't al versmaden. Daerom met banden // wil ickse versenden, Tot haerder schanden // met grooter ellenden, In vreemde landen, onder haer vyanden, Aenschout net enden. Och wat al plaghen // hongher pestilency, Mosten sy draghen // daer toe vyolency, So vele daghen // dat Godt mocht behagen, Haer penitency. Comt weer uyt Babel // nu alle ghelijcke, Acht niet als fabel // Gods woort en sijn rijcke Weert den diabel // weest niet variabel Maer autentijcke. Hier condy sporen // tweederleye leven, Dat was verlooren // ghinck Godt eerst verheven, Daer na met toren // ginck hys al verstooren, Die quaet bedreven. Tot alle beyden // lieven ende vreesen, Laet u toch leyden // door singhen en lesen, Wiltse recht scheyden // u oock recht bereyden Tot 'sHemels wesen. Eermen u dringen // wilt van liefde loopen Ter bruyloft ringen // wilt gheen ossen coopen Noch onschult bringen // van tijtlijcke dingen T'eewich wilt hopen. Recht wort ghepresen // lieven ende minnen, Laet zijn geresen // haer vruchten van binnen, Sy sal ghenesen // al uwe vreesen, Ghy sult verwinnen. Wel hen die't vaten // in een goe conscienty, Tijtelijck verlaten // doende penitenty, Haer leven haten // t'sal haer vele baten, In Gods sententy. Eenen Eleaser // seven sonen tellen, Eenen Beltsaser // met sijn drie ghesellen, Nebucadnaser // Antiochus dwaser, Woudense vellen. Doot nochte Helle // can die liefde crencken, Stroomen rebelle // canse niet verdrencken, Al woudy telle // al u goet daer stelle, Daer voor toe schencken. Want ellick strijder // worstelt in sijn percken Daer salmen yder // loonen na sijn wercken, Quade met lijden, goede met verblijden, Och wilt toch mercken. Waerom wilt hooren // des Apostels raden Laet niet verlooren // den tijt der ghenaden, Of Godes tooren // na t'gheclanckx labooren, Soud u veel schaden. Eer dese dingen // comt die toecomen moete Wilt doch gheringhen // doen oprechte boete, So suldy singhen // en van vreuchde springen, Dit is mijn groete.
