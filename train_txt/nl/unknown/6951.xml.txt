 AStrea soete Meyt Die uwe schaepjes weyt Door t'over-breede Klaver-gras, En daer de beeckjes klaer als glas Door vloeyen // besproeyen // het vruchtbaer lant Van d'Amstels groene kant, Hoe jeuchdich // hoe vreuchdich Wandelen wy nu tsamen hant aen hant. k'Weet gheen Goddin die leeft Die soo veel schoonheyts heeft Als mijne Son Astrea doet: O! Harderinne van mijn ghemoet Het grafje // vervarsje // door t'luchtich gaen, De soete Klaver-blaen. Begroeten // u voeten En gaen van vreuchde recht-op staen. Siet hoe het Pluym-ghediert Rontom ons henen swiert, En van ons tween een Lietgen singt: Ay: siet hoe t'weelighe gaytgen springt, Het schaepje // sijn knaepje // zijt wel ghemoet Seyt: Paren zijt ghegroet, Mijn Harder // waer varder? Niet als een teycken ghy mijn daer van doet. Op dit heuveltge neer In t'aenghename weer, Astrea lief, u leden rust, En laet u treurich oogh na lust Aen d'heyden // gaen weyden // door t'soete Vee Van Trijn Wouters dree, Gaet streelen // en deelen Malkander haer soete kusjens mee. De Son sijn aessem scheyt, Gheen blaetgen hem verreyt, Het vlieghent wildt is moe en loom, En t'Aemsteltge leyt onder stroom, t'Wort duyster // stil luyster // daer is ghewach: Dus Sela-dongh ghenach Seyt Astree // en hier mee Liet sy my daer alleen vol droef gheklach.
