 MInsaem beeldeken amoureus // Die t'heyrcracht overwint, Troostbaer herteken gratieus // Ghy zijt een seedbaer kint, Gemint // en wtgelesen // Int wesen // geen als ghy // Daeromme // Reyn blomme // Ick comme // tot dy // Mijn plagen // en clagen // Helpt dragen // met my. Aerdich, edel, besneden // rijs, Gesproten magnifijck, Hester zydy in seden // wijs, Waer is doch ws gelijck? Aertrijck // vervult dijn fame, Eersame // maget smal: O puere, Figuere, De fluere // van al, O schoone Persoone, De croone // zijn sal. Reyct my heden dijn witte // hant, Dus lange niet vertouft, Mijn teer herte deur hitte // brant, D'welck meer dan troost behouft: Bedrouft // ligh ick ter neder, Keert weder // soet aenschijn: Oorsake? Dijn sprake, Heeft smake // soo fijn: Dat heden, Dijn reden, Gebeden // moet zijn. Is bewesen sulck vrouwen // recht, Aen my, in voller macht, Dat ick uwen getrouwen // knecht, Ben aengenaem verwacht: Den nacht // is dan verdwenen, Vaert henen // valsch crackeel: Geen perten En smerten, Twee herten // geheel, Met sinnen Wy winnen Der minnen // casteel. Adieu, minnelick // cordiael, Jonck herte, heusch, en vry: Die goddinnelick // teenemael Mijn vreught zijt even by, Soo bly // als wy nu scheyden, Verbeyden // wilt my saen: T'onvroede Int goede Ten bloede // neemt aen, Houdt stede, In vrede, Hier mede // wy gaen.
