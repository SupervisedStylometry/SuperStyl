 ELck claeght dat hy // niet onbecommert heeft, Vrolijck en bly // den gulden tijt beleeft, Dies op het pas // al zijn wy vol gheweens, Is als sy was // maer t'volck is niet alleens. Tot nootdruft cleen // heeft dat Volck haer gevoecht, Daer nu elck een // aen overvloet misnoecht, Geeft dan de schult // den tijt niet maer t'gemoet, Dat niet vervult // en wert in overvloet. Was in de mensch // begeerlijcheyt noch niet, Hy had sijn wensch // als inde gulde tijdt, Nu leven wy // in vol op noch bedroeft, Maer t'comt daer by dat elck te veel behoeft. Die dan gherust // wil leven sonder haet, Soecke sijn lust // in een ghemene staet, Int heerschen niet // van anderen t'ghebien, Dat men doch siet // ghehaet altijdt gheschien. Doch die van Stof // is, dat hy gheern heerst Dwingh niemant of // bedwinght sich selven eerst Want t'sal ghewis // niet eender beter gaen, Voor dat elck is // sijn eyghen onderdaen.
