 Broeders goet, Grijpt nu een moet, En wilt hem loven, Wat hy ons doet // De Heere soet: Al van hier boven Prijst hem // O Ierusalem En wilt toch singhen Van vreuchde wilt ontspringhen. Hopt op de Stadt // Ick segt u plat En wilt niet vresen, Die den Enghel mat // Hy seyt ons dat Wy sullen daer wesen Is’t dat wy // Met herten vry, Op hem betrouwen Dan sal hy sijn beloften houwen. Laet ons gaen jent // ‘t Is ons bekent Want wy sullen hooren, Komt excelent // In mijne Tent Ghy zijt Herbooren, In mijnen Gheest // Vry onbevreest, Wilt toch toe treden Ick heb voor u ghebeden. Vrienden betrout // Toch hier op bout En wilt niet slapen, De straten van Gout // Ghy zijt dat zout, Wilt vreuchde rapen Hout u doch reyn // Ghy hoopken kleyn Wilt toch wat lijden, Want wy sullen ons verblijden. Daer en is geen nacht // hebt hier op acht Wilt u bereyen, Ghy kleyne geslacht // U toch wel wacht Ghy en sult dan niet schreyen, Noch gheene leet // Maer ‘t blinckende kleet Sult ghy antrecken, Met sijn mantel sal hy ons decken. Reynicht u toch // Van al bedroch, Daer en magh geen vuyl binnen Ick seght u doch // En roept geen och Hoe sullen wy ‘t winnen, Wie ‘t al verwint // Dit wel versint Die sal daer komen, Weest toch vrolijck, O ghy vromen. Met groot jolijt, Ghebenedijt Sullen wy hem dancken, Al die hier lijt // En met hem strijt, Die zijn sijne Rancken, Weest niet versaecht // Het kruys toch draegt En wert niet moede Want het Lam sal ons behoede. Die Son seer klaer // En dit is waer Sal niet meer schijnen, Ghy kleyne schaer // Tocht niet een haer En sal verdwijnen, Doet al uwen loop // Tot desen hoop Hy sal ons Kroonen En ons sijn vreucht verthoonen. En laet ons al // Met bly gheschal Toch hier op zayen, De wech is smal // Hoort dit nu al, Dan sult ghy mayen O Israel // Bereyt toch wel, Al uwe Lampen Om dese vreucht so laet ons kampen O Heer verhoort // Dit soet accoort Van u gheslachten, In de nieuwe geboort // Laet ons gaen voort Dat wy ons wachten, Van alle quaet // Komt ons te baet Wilt ons bewaren Op dat wy by u vergaren. O Prince Godt // Dit is het slot Daer wy nae haecken Want u ghebodt // Dient niet bespot, Al gaen sy ‘t laecken, Sal ‘t langer staen // Dat sy dus gaen U woordt vertreden, O Heere draecht sorge voor u leden.
