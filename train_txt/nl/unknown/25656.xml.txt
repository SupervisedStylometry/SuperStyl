 Int Hemelrijc // daer Christus die sal wijsen Die hem gelijc // hebben gedient voorwaer, Met daet en blijck // in cleeden, drincken, spijsen Sonder beswijck // in allen nooden swaer, Nu hier en daer // aen die cleyn Christen schaer Is vreucht, soo daer gheschreven staet, Die hy den sijnen gheven gaet, Hoort allegaer // van deser vreuchden claer. Af nu gheleyt // den doot, en van het sterven Versekertheyt // nu van den tweeden doot Verlost, bevrijt // het leven te beerven Eeuwich voorseyt // van den swaren aenstoot Wt aller noot // verlost uyt liefden bloot, Want die doot is verslonden snoo, Als sy dit wel deurgronden soo, Verheucht devoot // soo wordt haer blijschap groot. Comen dan voort // uyt alle p’rijckels nooden S’duyvels verstoort // en van der hellen pijn, Om met accoort // Godt te loven der gooden, Die naer sijn woort // haer heeft v’lost devijn Dit sl den schijn // van haerder vreuchden zijn, Verlost uyt s’vyandts tanden fel Door Christi stercke handen snel, Dees medicijn // verhooght haer t’herte fijn. Oock sullen sy // vrolijck zijn en v’hoogen, Van herten bly // om datse binnen gaen Ter plaetsen vry // daer sy g’lucksalich mogen Eeuwelijck by // die uytvercoren staen, Godt wel gedaen // eeuwelijck schouwen aen, Och wat vreuchde comt daer ontrent Boven int Hemels claer convent Naer Schrifts vermaen // die sy sullen ontfaen. Blijdlic v’jeucht // sal haer hert zijn ontsteken Heel wel v’neucht // om datse Christum schoon Vol aller deucht // sullen sien onbesweken, T’vermeerdert haer vreucht // boven in s’Hemels throon, Die haer ydoon // op set des levens Croon, Palmtacxkens vyerich groeyende In haer handt tierich bloeyende, Dit gheeft tot loon // Christus Iesus Gods Soon. Treuren vergaet // die blijschap sal v’meeren Als sy t’ghelaet // der Enghelen ghetal Hebben te baet // by den Heere der heeren Vroech ende laet // tot gheselschap voor al: Vrienden hoe sal // daer vreuchd’ zijn met geschal, Als sy by alle vromen hoort, En heylighen daer comen voort Van’t jammerdal // bevrijdt van ongheval. Op datse niet // naeckt en bloot en v’toonen Godt haer aensiet // het tweevoudighe cleyt Des lichaems siet // eerlijc by hem te woonen Daer over giet // hy nu die salicheyt, Maeckt ooc bereyt // t’verciersel uytgebreyt Der zielen // d’welck is hovelijck, Gherechticheyt seer lovelijck, All’ dit voorseyt // bringt blijschappe planteyt. Noyt smaec soo soet // noch lieflijcker aenschouwen Dan ons ghemoet // met onsen Godt certeyn Vereenicht vroet // sal zijn met vast betrouwen Godt met ons goet // die heylighen gemeen, Alle groot en cleen // worden te samen een ‘Met ons, en wy met haer, en wel Sal men daer op het snaren-spel Des herten reyn // spelen in s’Hemels pleyn. Int hemels lant // seer vast met God v’bonden Der liefden bant // ons soo met hem verbint, Aen elcken cant // sal dees liefd’ t’allen stonden Seer abondant // wassen als sy begint, Seere bemint // staet dit niet in den wint, Om gaen tot dese rusten in Keert daer toe uwen lust en sin, Wordt soo gesint // t’is t’beste dat men vint. Sluytlic hier me // vrienden wilt doch gedincken In s’hemels ste // te comen, op den padt, Maect u doch ree // dinct hoemen daer sal blincken Witter dan snee // ja als der Sonnen rat, Dinct vrienden wat // grooter vreuchden is dat Wie machse doch uytspreken hier, T’verstandt moet hem ontbreken schier, Soect desen schat // boven in s’hemels stadt.
