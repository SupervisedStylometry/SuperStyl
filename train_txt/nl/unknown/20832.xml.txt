 MYn aengenamen vrindt is doodt; Ach! roep uyt // overluydt, Ende segh // hy is wegh, Uyt ons' oogen is hy vervoert. Sijn lichaem is van ziel ontbloot Wonder snel // wonder snel // Godes wil Wou dat hy neder vil, Ons hart is seer ontroert. Hy weynich tijd was kranck geweest, Ach! niet meer // als den Heer Hem ontbood' // door de Dood Uyt dees werelts bedroefde baen. Wie sal niet wesen dan bevreest, Als haer pijl // en haer bijl // soo dree set De Dood, die op ons let, Die niemand kan ontgaen. Daerom soo oeffent u in deughd Ghy u all' // op dit dal, Eer dat komt // en verdomt Om uw' sonden, u, God den Heer. Dan sult ghy wesen g'heel verheught, En verblijdt // als den tijd // komen sal Dat binnen 's Hemels wall' Ghy komen sult met eer. Hy was een man seer kloeck en vroom Soo van re'en // als van le'en, G'lijck bekent // tot het end Hy geweest is aen yder-een: Soo dat het scheen te zijn een droom, Dat ons vrind // als een wind // als een roock Uyt dese werelt doock, En haestelijck verdween. Soo dan en kan geleertheyd niet Ons de noodt // die de Dood Ons brengt aen // doen ontgaen; Want de Dood yder-een vertreedt. Daerom ick bidd' u, voor u siet, En betrouwt // noch op gout // noch op kracht U oyt; noch oock op macht; De Dood komt eer ghy 't weet. En g'lijckerwijs ront omme-gaet Satan staechs // 's nachts en daechs, Om den mensch // na sijn wensch Te vervoeren na d' helschen brand: Soo oock de Dood' noyt stil en staet, Maer sy tracht // dach en nacht // in 't gemeen Te Moorden yder-een, En 't halen uyt dit Land. Soo dan geluckigh is die seer Die in strijd // dien hy lijdt, Dien hy heeft // niet en sneeft, Die ten allen tijd hout goe wacht. Des toe siet, vrinden, slaept niet meer In u quaedt // maer verlaet // uwe sond, Ey! doet toch dit terstond, En na de deughden tracht. O vrinden! dit wordt u geseyt Van die geen // wiens gesteen Na u jacht // ende tracht, Om dat ghy daer na leven soud. Ach! ach! uw' sonden hier beschreyt, Soo sult ghy // eeuwich bly // zijn by God: 'k Wensch dat ghy sijn gebod In alles onderhoudt. Bidt dat ons' vrind haest vry mach zijn Van 't gesteen // van 't geween, Van 't verdrijt // dat hy lijdt Om sijn sonden in 't Vagevyer: Dan sal verlost uyt all' sijn pijn Onsen vrind // die ons mindt // bidden we'er Ons algemeenen Heer, Dit 's d' Hemelsche manier.
