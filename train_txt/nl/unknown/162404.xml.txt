Recht heeft die preker schreven, Van een // met gheen // ghepaert, Wee hem in alle sijn leven, Die 't goet // met spoet // vergaert, Beter is hy verclaert, Een paer // can haer // verwarmen, Sijn vrient // verlient // hem d'armen So hy struyckelt vervaert. Ach wee is swaer om lesen, T'welck hy // brengt by // int perck, Wedenaers Weduwen Weesen, T'begin // den sin // aenmerck, T'is wee t'sy leeck of clerck, Een wort // wel cort // vergaende, Twee souden // t'houden // staende, Een drievout snoer is sterck. Niet alleen voor den Vader Door deuchd // dees vreuch // de rijst, Maer d'enghelen alle gader, Van een // onreen // die grijst, Want quaet en boet bewijst, Lucas // schrijft ras // van desen, Vijftien // wilt sien // en lesen, Tot hem keeren hy prijst. Roert doch die Schrift van wijven, Daer meest // Gods gheest // gaet raen Der testamenten drijven, Hoe 't oud // noch soud // vergaen, Maer t'nieu soud eewich staen, Sara // sprack dra // drijft schoone Wech gheeren // dienst deeren // haer soone, Die en sal gheen erf ontfaen. Wijven zijn d'instrumenten Gheweest // self leest // Gods Wet, Daer hy sijn Testamenten Voorwaer // seer claer // mee set, D'eerste was Lea met, Ses soon // heel schoon // beschoncken, Rachel // seer wel // quam proncken Met Ioseph dien hielp bet. Tot verlossing wert sonden Den soon // van schoon // Rachel, De Sone Gods bevonden Een hooft // danckt looft // hem wel, Van 'tgheestelijck Israhel, Sijn lust // vry blust // in deuchden, Sijn wil // houdt stil // in vreuchden En leeft na sijn bevel. Daer staet Remeynen ellef, V houdt // niet stout // maer vreest Want die natuerlijcke sellef, Hoe claer // elck waer // die meest, Zijn niet verschoont gheweest, Veel schilt // wy wilt // doch waren, Ons condt // goet rondt // niet sparen Dan door een ander gheest.
