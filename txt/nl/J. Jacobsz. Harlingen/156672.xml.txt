 ICk belijt // het met vlijt // op een tijt My hebben met reen // been, Twee verblijt, om een gave cleen, Op dat sy't ghebruyckten ghemeen, Eenen sanck // wast begheeren, Ick dorstet lanck // niet consenteeren, Hierom want // mijn verstant // alle cant Ghelijcke veerdich niet // siet, Nader hant // ick my noch beriet, Ende vant // dit ghetuyghe liedt. In de leer // die de Heer // sijn volck teer, Heeft tot een ghebedt // set, Haer begheer, vervullende net, Eyscht hy eer // van elck onbesmet, Namelijck dit // segt te gader, Als ghy bidt // onse Vader, Ghy eersaem // uwe faem // ende naem, Gheheylicht alle tijt // zijt, Onbequaem // hen hier in verblijt Sonder schaem, die gantsche werelt wijt. Dese gront // wert vercondt // door die mont Van des Heeren boon // schoon, T'aller stont // ons tot eenen loon, Om ghesont // crijgen t'eeuwich loon, Israhel // Gods lof songhen, Doen sy snel // eerst ontspronghen, Door die zee // alletwee // twelck haer dee Pharao doen by gram // quam, Ende mee // haren wegh aennam, Wijt en bree // gantsch te grond hy swam. Tot dees leer, // tot Gods eer // noch veel meer Int Psalm boeck vervaet // staet, Alle weer // Davidts gantsche raedt, En begheer // alle hier op staet, God te prijsen // groot te maken, Te bewijsen // in veel saecken, Hoe verheucht // watte vreucht // ende deucht, Men in desen oort // spoort, So hem veucht // hier teghen discoort, Die lust der jeucht // dempt met s'heeren woort. Seraphin // Cherubin // oock niet min, Haer om sulcx te doen // spoen, Haren sin // strecket ons ten goen, Haer ghewin // is al ons behoen, Ellick veylich, roept om seere, Heylich heylich // heylich Heere, Onse Godt // Sebaoth // u ghebodt, V glorieuse raet // gaet, Ellick tot // deur en ellick straet, Hy is sot // die dit niet en vaet, Een yeder een // groot en cleen // die dees reen Wech uyt sijn ghedacht // slacht, Wat die meen // is niet en betracht, Comt in vreen // als elck Christen lacht Hoe gheschien // moet dit prijsen, Het bedien // sal u wijsen Met accoort // onghestoort // Godes woort, Matheus in het vijft // schrijft, Dese oort // ons in Christo lijft, Die t'hoort // ende daer by blijft. Niet toe laet // datmen quaet // inder daedt Van u met waerheyt // seyt, Altijt staet // haer tot deucht bereyt, Die u smaet // aendoen ende leyt, Goet te wenschen // haer te stichten, Voor die menschen // laet so lichten Vwen ganck // datse danck // vry en vranck, Segghen onsen Heer // seer Met verlanck // volghen uwe leer, Sulcken clanck // gheschiet God ter eer. Daer toe bly // bidden wy // t'allen ty, Spreeckt Paulus den Heer // seer, Op dat hy // u te recht bekeer, So dat ghy // weerdelijck sijn leer, Wandelen sult // al u daghen, En vervult // na sijn behaghen, Met goetheyt // dat gheseyt // en verbreyt Zy door u sijn prijs // wijs, Sijn Majesteyt // heerlijcke logijs, Wel bereyt // die in deuchden grijs. Eenen raet // schreven staet // selver gaet Iohannes vijfthien // sien, Na die maet // moetet recht gheschien, T'alligaet // sal ick u bedien, s'Vaders eer // wordt bedreven, Met begheer // als ons leven, Dat elck spoet // vruchten doet // overvloet, Te worden Christus knecht // recht, Hy behoedt // in alle ghevecht, Ons ghemoedt // hoe simpel of slecht. Al u werck // swack of sterck // leeck of clerc Al tot Godes eer // keer, In elck perck // vordert sijnen leer, Dat gheen swerck // v duyster verneer, So ghy u leven // na t'vleysch wenschen, Gaet begheven // als andere menschen, Ezechiel // waerschout wel // dat ghy snel, Maeckt des Heeren naem // blaem, V ghesel // daer door comt tot schaem, Die werelt fel // lastert onbequaem. Eer ick sluyt // dat beduyt // hoort my uyt, Maeck ick tot u tween // reen, Dit gheluyt // vervult u ghebeen, Gheen menuyt // of het zy ghemeen, Mijn begheer // is t'versieren, Van Gods leer // in alle manieren Dese raet // uwe staet // mede vaet, En die een ander dient // mient, Paulus laet // Titum sijnen vrient, De maet // weten niet verstient. Valsche schijn // gheen termijn // latet ijn, V woort of werck raet // daet, Al ist u pijn // immers oprecht staet, Oftser zijn // oft niet die 't aengaet, Denckt hier by // God ick diene, So sal hy // ghedult verliene, Ghehoorsaem // trouw bequaem // aenghenaem V haer die u loont // toont, Oftse u naem // lastert ofte hoont, Dat al tsaem // immers niet verschoont. Mijnen sanck // neemt in danck // elc so lanck Dat hy hem verhuert // duert, Vwe ganck // onderdanich stuert, Dat gheen stanck // Gods naem versuert, Vwe Heeren // oock d'onaerdich, Alder eeren // achtet waerdich V verblijt // als ghy lijt // in dees tijt, Om gheen misdaet // quaet, Oftme u smijt // en daer toe versmaet, Sulcken strijt // is voor God ghenaet. Even wel // niemant sel // sijn rebel, Dien God inden gheest //vreest, Maer seer snel // voldoen aldermeest, Haer bevel // self die Schrift deurleeft, Mijnen liedt // of mijn praten Achtet niet // maer wilt vaten Petrus leer // Paulus meer // God ter eer, By dat elck u schrijft // blijft, Mijn begheer // dan ghenoech beclijft, Looft den Heer // bidt dat hy u stijft.
