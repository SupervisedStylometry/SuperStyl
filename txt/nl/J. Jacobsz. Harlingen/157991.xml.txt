 Hooch en duer // ist bestuer // van het vuer Levitici thien // t'sien Tot figuer // moestet doen gheschien, Maer heel puer // is den sin van dien, Liefde is het ware wesen, Some Canticorum lesen, Nemet merck // sy is sterck // in haer werck, Ghelijck doot en hel // fel Houdt haer perck // teghen veel ghequel, Leeck of clerck // maeckt haer niet rebel. Een gheleert // hooch gheeert // die temteert, Christum onder 't schijn // fijn, Als begheert // onderwesen t'zijn, Maer verkeert // t'hert is vol fenijn, Hy en was niet verslaghen, Ginck nochtans Christo vraghen, Wat ghebodt // t'hoochste lot // heeft van God T'welck is in de Wet // set, Dese sot, onghereghelt tret, Christus tot // hem antwoorde net. Niet allien // ses en thien // maer ick mien Oock in 't dertichst plaen // t'staen T'vijfde boeck // deed' Christus vermaen, Sijn versoeck // die 't recht can verstaen, Lief te hebben God u Heere, Hert en siele daer toe keere, Oock 'tghemoet // hy is goet // die 't so doet, Dit ist grootst bevel // wel, T'ander soet // dit ghelijcken sel, D'naesten moet // ghy oock beminnen snel. Dese twee // gants gheree // daer ter stee, Van Christo belast // vast, Wel ter snee // worter aenghepast, Wet ende mee // der Propheten wast, Ditte gaet ons onderwijsen Dat die liefd' is hooch te prijsen Wie misdoet // uyt vleys oft bloet // om tijtlijc goet, Oft om anders wat // schat, Is onvroet // boven Gode dat, T'sijn onspoet // heeft hy lief ghehadt. Raet nu voort // met accoort // soo't behoort Die dees liefd' uyt twist // mist, Die verloort // al daer hy op gist, Telcker oort // oft hy wel al wist, Tonghen ende Prophecyen, Niet sal hyder mee bedyen, Geloove vaet // wonder daedt // hoort oft laet Sonder liefd ist niet // siet Elcker maet // t'goet den armen biet, Oock't ghewaet // t'lichaem branden liet. In haer soet // is ootmoet // vrientschap goet Sy toont niemant spijt // nijt, Schalckheyt doet // sy tot gheender tijt, Noch verwoet // opgheblasen crijt, Sy bedriecht in ghenen hoecken, Sy en gaet t'haer niet soecken, Niet verwacht // verdriets clacht // erch ghedacht, Noch dats' om tweedracht // lacht, Maer met kracht // die waerheydt betracht Dach en nacht // tdoet haer wee of sacht. Claer bescheyt // hier verbeyt // als gheseyt Wordt in Moyses Wet // net, Alle beyt // Gods en s'naesten met Oock bereyt // u sin hier toe set, Siedy s'broeders vee gaen dolen Exo.23,4. V wordt neerstelijck bevolen, Niet ontreckt // u suspeckt // maer perfeckt Tot u broeders deer // keer, Isset sect // ofte valsche leer Niet begect // maer helpet doch weer. Toont nu klaer // met gebaer // hier en daer, Dat in u Godt bemint // vint Is niet waer // s'werelts liefd' ontrint In u maer // zijn ghebodt versint, Die hier s'werelts vriendt wil wesen Sal Gods vyandt zijn in desen, Wat ghetier // oft manier // dat men hier Stellen tot een toon // schoon Voor het vier // gheltet met een boon Hert en nier // doorsocht elck zijn loon. Isser nu // vrees en schu // dat ghy u Voor het kruys versaecht // draecht, Al ist ru // en u seere plaecht, Sulcks verdu // ende van u jaecht: Iacob schrijft wilt verblyden Als ghy veel becoring lyden, Dit versint // wie hem vint // een Gods kint Sal voor gheen kastien // vlien, Troost ontrint // isser t'allen tien, Wie verwint // sal veel goets gheschien. In den Heer // stelt nu teer // u begheer Om der liefden werck // sterck Langs soo meer // te brenghen in't perck, Esa.58,8. Nae zijn leer // tot stichting der kerck, Hy verbiet // overspelen, Doodtslaen siet // ende stelen, Ghetuyghe vals // geen van als // dies getals Nimmer doch begheer // meer Myr of t' als // zijnde bitter seer Met regals // neemt al hier zijn keer. Sonderling // in dit ding // soo ick sing, V naesten wel ghesint // mint Gantsch ghering, oft ghy 't selver bint, Sonder dwing // als die eerst begint: Die liefde doet niet ten quaesten Ymmermeer zynen naesten; Maer ghehult // zy vervult, alle schult Van der gantscher Wet // net, Met ghedult // u toch hier nae set En ghy sult // dan zijn onbesmet. Salicheyt // is toeseyt // met bescheyt Niet die alleen t'woort // hoort Maer bereyt // tottet werck accoort, Twellick leyt // inder liefden oordt, Liefde is die hooftsomme Der Gheboden Gods alomme, Hoort ghemien // t'kan niet schien // dan allien, Alst een reyn hert // wert, Ende vlien // dat conscientie smert, Oock by dien // int gheloof volhert. Och wat raedt // menich quaet // van dees maet, Hebben veer ghefaelt // dwaelt Onnutte praet // by haer toe ghehaelt, Sonder baet // cierlijck af ghemaelt, Als leeraers die wet vertellen, Verstaen niet watse segghen oft stellen, De Wet is goet // tot behoet // van 't ghemoet, Ghebruycktse yemant recht // slecht, Maer men moet // weten die oprecht Is seer soet // gheen Wet opghelecht. OP dat dit // elck lit // onbesmit, Doch in rechter maet // vaet, Neerstich bidt // den Heer om ghenaet, Hier in sit // ledich vroech noch spaet, Laet u leven claer so blijcken, Datter elck op mach kijcken, Metter cracht // met ghedacht // houdt goe wacht Op dies' Heeren gheeft // vreest Dit gheslacht // een goet voorbeelt weest, Inder nacht // gaen die menschen meest. Neemt in danck // desen sanck // gaet niet manck, Al waer schoon die noot // groot, V verlanck // zy in elck aenstoot, Eer yet lanck // na een ander boot, Comt nu ghy ghebenedijde, Die bocken aan de lincker sijde, Leert dit voort uyt Godts woort // met accoort, V selfs onverschoont // toont, Onghestoort // wort ghy dan beloont, Wie u hoort // wort met u ghecroont.
