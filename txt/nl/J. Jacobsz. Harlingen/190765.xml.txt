 TOt maechdelijcke spruyten Mijn dicht // ick richt // eersaem, Die binnen ende buyten, Aen gheeft // oock leest // lichaem, Heylich reyn en bequaem, Oock meer // den Heer // te achten, Dan t'recht // van d'echt // betracht, Dit leert Paulus te saem. Ick wensch u voor een groete, Veel deucht // o jeucht // verlicht, Ghenade vrede boete, Een waer // een claer // ghesicht, Een wandel wel ghesticht, Bedacht // dach nacht // met smeecken, Voort val // der smal // ghebreecken, Dat doch t'ghemoet niet swicht. Siet dese salutacy, Tot goet // met spoet // om meest, Zy u van Godes gracy, Ghewracht // met cracht // des gheest, Door Christum onbevreest, Op dat // ons schat // gheladen, Verdriet // gheschiet // noch schaden, Die Schrift doorsoeckt en leest. Ghelijck ghy zijt natuerlijck, Elck maghet // waghet // sacht, Met Gods Heylighen puerlijck, Dat ghy // en wy // veracht, Maechden zijn inder cracht, Die t'Laem // vreetsaem // navolghen, Noch list // noch twist // verbolghen, Moet zijn van ons bedacht. Eerstelinghen vercoren, Geen goet // maer bloet // ghecost, Kinders weder gheboren, Die elck // met melck // en most Ghelaeft zijn en verlost, Dees prijs // als wijs // van moede, Dees naem // bequaem // en goede, V neme brood' noch post. Roem van naem of waerdye Niet vaet // is daet // absent, S'gheestes of vleysch hoverye, Slaet doot // hoe groot / present God bewaer ongheschent Die recht // en slecht // hier leven, Met vreucht // tot deucht // haer gheven In blyschap en torment. Reyn heylich van manieren, Sulck maechden // haechden // t'lam, Die haer met deucht verchieren, Niet stout / met gout // aen quam, Noch onrechtveerdich mam, T'gheslacht // hier sacht // van sinnen, Dit stil // Gods wil // beminnen, Zijn vanden rechten stam. In Exody // wy // leeren, Tien paer // wel claer // subijt Vader en moeder t'eeren, Ephesen // lesen // wy't Sierach drie dreycht met vlijt, Die kinder // hinder // t'senden Die d'ouden // souden // schenden, Als nu in deser tijt. T'wort in Matheo vonden Ick mien // vyfthien // die vaet, In Marco oock dees gronden Int seven // schreven // staet, Collossen drie oock gaet, Die meer // begheer // om weten Besie oock die Propheten Wat hier is Godes raet. Spieghelt hier in u wercken, T'is wel // t'bevel // van God, Maer weynich die 't bemercken, Met hoopen // loopen // sot, Die jeucht acht Gods ghebodt Voor niet // besiet // haer ganghen, Sy dichte // lichte // sanghen, Dit is des werelts rodt. Sara in der stadt mede, Sie seyt // bescheyt // aenhoort, Met die ydel van zeden, En meen // ick gheen // accoort, En die Psalmist opboort, Wel dien // die vliet // den quaden, Altijt // met vlijt // beladen, In Godes heylich woort. Oock heeft Salomons leeren Een schoone // croone // toe seyt, Die d'ouders obedieren, Der looser // booser // seyt, Gheensins en zijn bereyt, Door woeden // spoeden // waerlijck, Tot quaeden // schaden // vaerlijck, T'verderven haer verbeyt. Och waer gaer ick my keeren, Wat saecke // maeck // ick hier, Met vermanen en leeren, Diens vaer // om swaer // dangier Te weeren, heeft regier, Sijn huys // t'abuys // te letten, Gods gheest // dits meest // ginck setten, Hem over zijn bestier. Neemt my doch dit ten besten, Want ick // verschrick // aen een, Nu ick bedenck ten lesten, Tot wien // gheschien // mijn reen, Want mijn wijsheyt is cleen, Gheen const // maer gonst // en minne, Wt richt // en dicht // den sinne, Denckt my in u ghebeen. Ick was sorchd ick verbonden, V cloeck // versoeck // sulcx dee, Door u laetste vermonden, T gheschenck // ghedenck // ick mee, Voort liefde trou en vree, Die drijven // t'schrijven // t'samen, Maer siet // t'gheschiet // met schamen, Hoewel met danckbaerhee. Die dochters die ick meene, Haer vaer // wort claer // ghenaemt, Eerste letters by eene, Elcx vers // so dwers // versaemt, Tys Gherritsen het raemt, Waer door // ter coor // bedacht // siet, Met haest // het naest // die clacht // schiet, Memory wort gheblaemt.
